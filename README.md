# francesca-liddy-bot
Bot no Telegram com persona de IA (Francesca Liddy de Better Call Saul), construído com n8n, OpenAI e Google Workspace. Automatiza geração de documentos PDF, extração de dados estruturados com IA, transcrição de áudio e busca de notícias em tempo real. Hospedado em VPS DigitalOcean com Docker + Ngrok.
